# CRUD NODE con Express y MySql


## Instalación
Necesitas instalar las dependencias, mysql y ejecutar el archivo schema.sql en la base de datos
```
npm install
npm install mysql
mysql -uroot -h127.0.0.1 < crud-node/schema.sql
```

## Ejecución
```
node crud-node/app.js
```
